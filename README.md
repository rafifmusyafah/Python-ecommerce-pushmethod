# Python-ecommerce-pushmethod
ini adalah project pertama saya yang diberikan oleh mentor saya sebagai data engineer

disini saya membuat sebuah aplikasi sederhana dengan python yang dimana saya diminta untuk menghitung total pendapatan dan total pajak secara akurat menggunakan bahasa python. 

saya menggunakan salah satu konsep OOP/object-oriented programming yaitu encapsulation. Secara sederhana, encapsulation adalah proses membungkus data (variabel) dan metode (fungsi) menjadi satu kesatuan unit yang disebut kelas (class), sekaligus membatasi akses langsung dari luar.


dan saya melakukan pengujian dengan memasukan 4 data penjualan dengan total pendapatan Rp.1.525.000 dan program berhasil menghitung pajak 10% sebesar Rp.152.500 secara otomatis.
